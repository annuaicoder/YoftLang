Welcome to Engine

Engine is an interpreted programming language focused on speed and simplicity. Designed for developers who want to write clean, readable code without unnecessary complexity.

Quick Start

1

Clone the Repository

git clone https://github.com/yourusername/Engine-Lang.git
cd Engine-Lang
2

Run Your First Program

python -m engine test.eng

Example Code

Variables & Arithmetic

var x = 10
var y = 5

show x + y
show x * y

Strings & Output

var message = "Hello Engine!"
show message

var name = "World"
show "Hello " + name

Conditional Logic

var score = 85

if score {
    show "You passed!"
}

var temperature = 25
if temperature {
    show "It's warm outside"
}

Infinite Loops

var counter = 0

FOREVER {
    show counter
    var counter = counter + 1
}
