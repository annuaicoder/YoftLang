The Yoft Programming Language

A compiled language that produces native binaries. Clean syntax, zero runtime dependencies, maximum performance.

Why Yoft?

Compiled & Fast

Compiles to native machine code through C. No interpreter, no VM, no overhead.

Zero Dependencies

The compiled binary is completely standalone. Ship it anywhere, run it everywhere.

Clean Syntax

Readable, minimal syntax. No semicolons, no type annotations, just code.

Module System

Import other .eng files to organize your code into reusable modules.

Installation

Requires Go 1.21+ and a C compiler (gcc/clang).

terminal
# Clone and build
git clone https://github.com/annuaicoder/Engine-Lang.git
cd Engine-Lang
go build -o yoft .

# Install globally
sudo mv yoft /usr/local/bin/

# Verify
yoft version

Quick Start

1. Write a program

hello.eng
var name = "World"
show "Hello, " + name + "!"

2. Compile & run

terminal
# Compile and run in one step
yoft run hello.eng

# Or compile to binary
yoft build hello.eng -o hello
./hello

Documentation

Everything you need to write Yoft programs.

Variables

Declare variables with var. Reassign with =.

var name = "Yoft"
var x = 10
var pi = 3.14
var active = true

x = 42         # reassign

Supported types: int, float, string, bool, list, null.

Output

show "Hello World"
show x + y
show "Result: " + str(x * y)

Operators

Control Flow

If / Else

if score >= 90 {
    show "A"
} else if score >= 80 {
    show "B"
} else {
    show "C"
}

While Loop

var i = 0
while i < 5 {
    show i
    i = i + 1
}

For Loop

for i in range(10) {
    show i
}

var items = ["a", "b", "c"]
for item in items {
    show item
}

Functions

func greet(name) {
    show "Hello, " + name + "!"
}

func add(a, b) {
    return a + b
}

greet("Alice")
show add(3, 7)

Functions support recursion, local scope, and closures.

Examples

FizzBuzz

for i in range(20) {
    if i % 15 == 0 {
        show "FizzBuzz"
    } else if i % 3 == 0 {
        show "Fizz"
    } else if i % 5 == 0 {
        show "Buzz"
    } else {
        show i
    }
}

Recursive Fibonacci

func fib(n) {
    if n <= 1 {
        return n
    }
    return fib(n-1) + fib(n-2)
}

for i in range(10) {
    show fib(i)
}
